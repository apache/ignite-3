# API Reference

Comprehensive API documentation for all supported languages and drivers.

## In This Section

### Native Client APIs

Full-featured clients with support for all Ignite APIs.

#### [Java API (PRIMARY)](./native-clients/java/)
The primary Ignite API with complete feature set.

- [Client API](./native-clients/java/client-api) - IgniteClient thin client
- [Server API](./native-clients/java/server-api) - IgniteServer embedded node
- [Tables API](./native-clients/java/tables-api) - Table, RecordView, KeyValueView
- [Data Streamer API](./native-clients/java/data-streamer-api) - High-throughput streaming
- [SQL API](./native-clients/java/sql-api) - SQL execution
- [Transactions API](./native-clients/java/transactions-api) - Transaction management
- [Compute API](./native-clients/java/compute-api) - Distributed computing
- [Catalog API](./native-clients/java/catalog-api) - Schema management and annotations
- [Criteria API](./native-clients/java/criteria-api) - Type-safe queries
- [Network API](./native-clients/java/network-api) - Cluster topology
- [Security API](./native-clients/java/security-api) - Authentication

#### [.NET API](./native-clients/dotnet/)
Native .NET client for C# and F# applications.

- [Client API](./native-clients/dotnet/client-api) - IIgniteClient
- [Tables API](./native-clients/dotnet/tables-api) - ITable, IRecordView, IKeyValueView
- [Data Streamer API](./native-clients/dotnet/data-streamer-api) - Data streaming
- [SQL API](./native-clients/dotnet/sql-api) - SQL execution
- [Transactions API](./native-clients/dotnet/transactions-api) - Transaction management
- [Compute API](./native-clients/dotnet/compute-api) - Distributed computing
- [Network API](./native-clients/dotnet/network-api) - Cluster topology

#### [C++ API](./native-clients/cpp/)
High-performance C++ client.

- [Client API](./native-clients/cpp/client-api) - ignite_client
- [Tables API](./native-clients/cpp/tables-api) - table, record_view, key_value_view
- [SQL API](./native-clients/cpp/sql-api) - SQL execution
- [Transactions API](./native-clients/cpp/transactions-api) - Transaction management
- [Compute API](./native-clients/cpp/compute-api) - Distributed computing
- [Network API](./native-clients/cpp/network-api) - Cluster topology
